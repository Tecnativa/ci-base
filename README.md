# CI Base

This is a simple repo for a simple image suited as to be a minimal package of
dependencies needed to run CI/CD workloads.

## What is inside

-   buildah
-   curl
-   docker
-   docker-compose
-   git
-   jq
-   pipx
-   podman
-   poetry + poetry-dynamic-versioning
-   pre-commit
-   python 3
-   skopeo
